## Инструкция по сборке
Выполнить следующую команду:

`mvn package`

Чтобы запустить программу после сборки в командной строке, нужно, находясь в папке target, выполнить следующую команду:

`java -jar "lab_1-1.0-SNAPSHOT.jar"`
